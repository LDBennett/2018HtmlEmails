Large Format Month!
Large Format Best Practices
  • Group jobs on the same material
  • Schedule site visits for installs
  • Plan a kick off meeting for large or "strange" jobs
  • Measure twice, print and install once
Avoiding Large Format Waste
  • Order boards to size
  • Cut substrate waste to ‘typical’ sizes
  • Have a clean bin for backing to use for packaging
  • "Sample" books printed in the wasted space
